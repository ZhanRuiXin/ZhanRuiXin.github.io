## 🚀 Local Development

To run this site locally:

```bash
# Clone the repository
git clone https://github.com/ZhanRuiXin/ZhanRuiXin.github.io.git
cd ZhanRuiXin.github.io

# Install dependencies
bundle install

# Serve the site locally
bundle exec jekyll serve -l -H localhost

# Open in browser
# http://localhost:4000
```

## 📝 Adding Content

### Add a Publication
Create a new file in `_publications/` with front matter:
```yaml
---
title: "Your Paper Title"
date: 2024-01-01
venue: "Conference/Journal Name"
citation: "Author names, etc."
---
```

### Add a Talk
Create a new file in `_talks/`:
```yaml
---
title: "Talk Title"
date: 2024-01-01
venue: "Event Name"
---
```

### Update CV
Edit `_data/cv.yml` with your latest experience and education.
